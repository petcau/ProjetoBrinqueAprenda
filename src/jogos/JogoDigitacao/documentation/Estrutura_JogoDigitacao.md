
## Jogo: Jogo da Digitação

---

### Estrutura de pastas:

```
JogoDigitacao/
├── componenetes/     # Componentes do projeto
│   ├── fases.json       # Json das fases do jogo
│   ├── Jogo.jsx         # Componente principal do jogo
│   └── Semaforo.jsx     # Componente que renderiza o semaforo           
├── CSS/
│   └── Semaforo.css     # CSS do semaforo
│
├── documentation/                 # Documentos do projeto
│   ├── ATA_Projeto_Jogo_Digitacao.md       # Registro das etapas do desenvolvimento
│   ├── documentacaoJogo.md                 # Documentação geral do jogo
│   └── Documento_Projeto_Jogo.md           # Documento do projeto principal
│
├── game_assets/                   # Recursos gráficos do jogo (fundos, ícones, feedback visual etc.)
│   └── FontesDigitacao/                   # Fontes usadas no design das palavras
│
├── Paginas/                       # Páginas principais do jogo
│   ├── Home.jsx                             # Página inicial do jogo
│   └── Digitacao.jsx                        # Tela principal do jogo de digitação
│
├── game.css                      # Estilos visuais do jogo
```
